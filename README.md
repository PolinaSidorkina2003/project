# Проект: Веб-ресурс по оказанию образовательных услуг
- Описание проекта: Веб-ресурс является обоазовательной платформой, предоставляющей услуги для учеников, сдающих ОГЭ и ЕГЭ по английскому языку. На моем веб-ресурсе ученики могут пройти тест, определяющий их уровень языка. По уровню языка подбираются преподаватели с индивидуальными наработками для учеников различных уровней владения иностранным языком с учетом их целей.
